This repository contains a collection of frontend projects built to strengthen my skills in "HTML, CSS, & JS".
Each project focuses on UI design, layout, responsiveness, and core web development concepts.

## Projects Included
1. Amazon clone website:
   A front-end clone of the Amazon homepage created using only HTML and CSS.
   
  **Features:**
  - Navigation bar with search functionality (UI)
  - Product sections and cards
  - Footer layout similar to Amazon
  - Clean and structured UI

2. Drum Kit: 
   An interactive drum kit built with JavaScript.

**Features:**
- Play drum sounds by pressing keys on keyboard  
- Visual button animations when triggered  
- Multiple sound effects mapped to different keys  
- Lightweight and responsive design  

3. Live Clock: 
   A live clock built using HTML, CSS, and JavaScript.

**Features:**
- Displays current time in real-time  
- Updates automatically every second  
- Clean and minimal design
